# Global Mental Health Depression Analysis Dashboard

## Project Overview
This Power BI project analyzes global depression trends and mental health patterns using multiple datasets.

- Power Query
- DAX
- GitHub

## Source Dataset (Excel)
- The original dataset used for this project has been included in this repository for educational and portfolio purposes.

## Key Insights
- Depression prevalence varies significantly across countries.
- Females generally show higher depression prevalence than males.
- Older age groups exhibit higher depression rates.
- Suicide and depression have a relationship but not a perfect correlation.
- Educational attainment may influence depression outcomes.
